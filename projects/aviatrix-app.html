Aviatrix App

Aviatrix is a publication created for professional women interested in aviation. It is a magazine for women of women. It features aviatrix from many different time periods and various aviation types. The project was created for a Publication class at Roger Williams University. The page animation follows the line, creating an interactive path through the articles.

Follow the line

The line is a subtle feature of the app. Thought it is just an extra design element, it provides a path allowing the user to traverse through the content. It guides the user through articles, galleries, and videos giving them a full interactive experience. Icons are provided showing the user which direction the next gesture should be directed.
